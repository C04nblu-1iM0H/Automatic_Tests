# Jest установка
```
$ npm i --save-dev jest
```
## Настройка package.json
Добавим в объект scripts новый ключь со значением запуск самого теста
```javascript
   "scripts": {
    "test": "jest"
  },
```
Это означет что мы можем при помощи команды (которая будет ниже) запускать наши тесты
```
npm test 
```
